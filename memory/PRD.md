# Herion - Digital Accountant Platform

## Architecture
Frontend: React + Tailwind + Shadcn | Backend: FastAPI + MongoDB | Auth: Cookie JWT | AI: GPT-5.2 via Emergent Key | i18n: IT/EN

- Catalog: 136 procedures, 7 categories
- Consulenza Rapida: AI triage (GPT-5.2) with confidence-aware UI
- Bilingual IT/EN system (sidebar, dashboard, login, register, welcome, profile, empty states)
- 3-Step Registration Wizard + Trust Layer (AES-256, GDPR, EU Data)
- Practice-to-AI Bridge: Context-aware chat within practices
- Tutela e Conformita: Rights/obligations/risks/sanctions per procedure

- Launch Polish: Signature moments, "What's happening" summary, completion experience, micro-interactions
