# Herion - Digital Accountant Platform

## Architecture
Frontend: React + Tailwind + Shadcn | Backend: FastAPI + MongoDB | Auth: Cookie JWT | AI: GPT-5.2 via Emergent Key | i18n: IT/EN | Responsive: mobile-first

## Responsive Breakpoints
- **< 640px (sm)**: Mobile — 2-col grids, stacked layouts, card-based lists
- **640-767px**: Small tablet — 4-col grids
- **768px+ (md)**: Desktop — sidebar visible, full table layouts
- **1024px+ (lg)**: Large — 6-col metrics, 2-column practice detail

## Key Features (Launch-Ready)
- Catalog: 136 procedures, 7 categories
- Consulenza Rapida: AI triage (GPT-5.2)
- Bilingual IT/EN system
- 3-Step Registration Wizard + Trust Layer
- Practice-to-AI Bridge: Context-aware chat
- Tutela e Conformita: Rights/obligations/risks
- Data Sovereignty: Privacy manifesto
- Creator Control Room + Father Agent
- Launch Polish: Signature moments, completion experience
- Fully responsive: mobile, tablet, desktop
